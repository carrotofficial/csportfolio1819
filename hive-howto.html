Flower task

The first task I created was to direct the bot to go to the flower with the most pollen. I used the following for-loop based structure to calculate said flower:

var mostFlower = game.flowers[0];

for (var i=0;i > game.flowers.length;i++) {
if (game.flowers[i].pollen > mostFlower.pollen) {
mostFlower = game.flowers[i];
}
}
